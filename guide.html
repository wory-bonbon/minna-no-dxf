基本操作

測定の基本

操作 方法 用途
沿い距離測定 線分をクリック → 測定開始 袖ぐり、衿ぐりなどカーブの実寸
直線距離測定 A点クリック → B点クリック 2点間の直線距離
幅/丈測定 幅/丈ボタン → 測定 水平・垂直方向の寸法
測定キャンセル ESCキー 測定中の操作を中止

※ DXF の作りによっては Shift+クリック の水平/垂直拘束が効かない場合があります。その際は幅/丈の数値を目安に確認してください。

測定値の見方

① 沿い距離（青太線）
カーブに沿った実際の長さ。袖ぐりや衿ぐりの実寸測定に使用。

② 直線距離（灰点線）
A点からB点への直線距離。

③ 幅/丈（橙L字）
水平方向の幅と垂直方向の丈。画面基準またはDXF座標基準で切替可能。

詳細機能

パーツ管理

  • ブロック（パーツ）ごとの表示切替
  • パーツ名の変更（ダブルクリックで編集）
  • ヘッダーの「↺パーツ / ↻パーツ / パーツ0°」で90度単位の回転
  • 設定は図面ごとに自動保存

測定ログ

  • 全測定履歴を自動記録
  • 部位メモの追加（股ぐり、肩線など）
  • CSV形式での書き出し
  • ドック表示/フローティング表示の切替（ログ▲/ログPOPボタン）

便利な機能

🔒 同一ポリラインロック

測定したい線分だけを選択可能。重なった線がある場合に便利。

🔄 全体回転 & フィット

ヘッダーの「↺全体 / ↻全体 / 全体0°」で向きを調整し、「Fit全体」「Fitロック」で素早く整列できます。

⌨️ ショートカット

Shift+クリックで水平/垂直制約、ESCでキャンセルなど。

活用のコツ

アパレル現場での使い方

🎯 袖ぐり・衿ぐり測定
沿い距離測定でカーブの実寸を正確に測定。パターンメイキングの必須機能。

📏 着丈・身幅測定
幅/丈測定で基本寸法を素早く確認。仕様書作成に便利。

📝 部位メモ活用
測定ログに「前股ぐり」「後肩線」などメモを追加して管理。

効率的な作業フロー

  1. パーツ名設定：最初に前身頃、後身頃などパーツ名を設定
  2. ロック機能活用：同一ポリラインロックONで測定対象を限定
  3. 測定実施：必要な箇所を順次測定
  4. ログ確認：測定ログで数値確認、部位メモ追加
  5. CSV出力：Excel等で活用
💡 おすすめ設定
「同一ポリラインにロック」をONにすると、測定したい線だけを選択できます。
重なった線がある場合に特に便利です！

書き出し機能

📊 CSV出力

  • 用途: Excel / Googleスプレッドシートで測定ログ管理
  • 対応環境: Windows / macOS の Excel 2019・2021・Microsoft 365、Googleスプレッドシート
  • ポイント: BOM付きUTF-8で文字化けなし。目盛り設定は不要です。

🎨 SVG出力

  • 用途: Illustratorでパターン修正・注釈を加えたいとき
  • 対応環境: Windows / macOS の Illustrator 2021 以降（Creative Cloud）
  • ポイント: 目盛りONなら20mm余白付き実寸ガイド、OFFでクリーンなアートボード。

📄 PDF出力

  • 用途: 実寸印刷で検品・寸法チェック
  • 対応環境: Windows / macOS の Adobe Reader・Edge・Chrome 印刷ダイアログ
  • 手順: 目盛りON推奨。印刷時は「実際のサイズ(100%)」「用紙に合わせない」を選択。

🖼️ PNG出力

  • 用途: PowerPoint / Keynote / Googleスライド等の資料貼付
  • 対応環境: Windows / macOS の Office 2019・2021・Microsoft 365、Google Workspace
  • ポイント: 目盛りOFFで資料に馴染む。正確な寸法確認はPDFを併用。

単位スケール設定

• ブラウザによって書き出し品質が異なる場合があります

セキュリティについて

データの取り扱い

📍 ローカル完結型の動作
このツールはお使いのブラウザ内だけで動作します。インターネット接続は不要で、外部サーバーとの通信は行いません。

🔒 データの保存場所
図面データや測定結果は、ブラウザのローカルストレージ（お使いのPC内）にのみ保存されます。クラウドへのアップロードは行われません。

📂 ファイルの処理
DXFファイルはブラウザのメモリ上で処理され、処理後は自動的に破棄されます。

よくあるご質問

❓ どのCADのDXFに対応？

一般的なアパレルCADが出力するDXF形式に対応。パターンのカーブはポリラインとして処理され、袖ぐりや衿ぐりの沿い距離を正確に測定できます。

❓ カーブの測定はできる？

はい、ポリライン形式のカーブは沿い距離測定で実寸を計測可能。ただし円弧（ARC）やスプライン（SPLINE）は直線近似で表示される場合があります。

❓ 機密図面を扱っても大丈夫？

図面データは外部に送信されることはありません。ただし、お使いのPCのセキュリティ状態もご確認ください。

❓ IT部門の承認は必要？

社内規定によって異なります。HTMLファイルの実行に関する規定をご確認ください。

❓ セキュリティリスクは？

公式リポジトリから入手すれば、一般的なセキュリティリスクは低いです。改変されたファイルにはご注意ください。

❓ データはどこに保存される？

設定や測定ログはブラウザのローカルストレージ（お使いのPC内）に保存。容量制限により古いデータが削除される場合があります。
